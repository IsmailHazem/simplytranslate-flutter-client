Это приложение является альтернативным клиентом для Google Translate, но оно не подключается к нему напрямую.
Оно работает следующим образом:
1. Текст, введённый вами (пользователем) отправляется на случайную инстанцию (сервер) SimplyTranslate.
2. Эта инстанция запрашивает перевод у Google Translate.
3. Google Translate отправляет перевод назад на инстанцию.
4. Инстанция отправляет перевод вам.

Таким образом, Google Translate не может определить вашу личность и собрать какие-либо данные о вас. Автоматический выбор случайной инстанции (режим по умолчанию) для каждого запроса перемешает и распределит ваши данные между инстанциями, поэтому ничего не сможет быть связано с вами так просто, так как это не централизованная база данных.

<b>Функции:</b>
* Поддержка 108 языков
* Текст-в-Речь (TTS или Text-To-Speech)
* Может получать текст из других приложений
* Кнопка перевода в меню выделенного текста
* Автоматический выбор случайной инстанции для каждого запроса
* Обновляемый официальный список инстанций SimplyTranslate
* Добавление пользовательских инстанций
* Тёмная тема
* Распознавание текста через камеру
